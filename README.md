## Features per Question

**Q1 – Balanced Parentheses Checker**

- Uses **stack** to validate parentheses in expressions.

**Q2 – Infix to Postfix Conversion and Evaluation**

- Converts mathematical expressions and computes result efficiently.

**Q3 – Reverse Traversal in Linked List**

- Traverses linked list in reverse order without modifying the original list.

**Q4 – Doubly Linked List**

- Supports **insert after a given node** and **delete node** operations.

**Q5 – Graph Traversal**

- Implements **undirected graph using adjacency matrix**
- Performs **BFS** and **DFS** traversal.

**Q6 – Min and Max Heap**

- Builds **min-heap** and **max-heap** from an unsorted array.

**Q7 – Dijkstra’s Algorithm**

- Finds **shortest path** from a source node in a weighted graph.

**Q8 – Random Integer Sorting**

- Generates **N random integers** and sorts using **Bubble, Selection, Insertion, or Merge Sort**
- Tracks **comparisons** and **swaps** (where applicable)

## How to Run

1. **Clone the repository**

```bash
git clone https://github.com/akumarce/dsa_assignment.git
cd dsa_assignment
```

2. **Compile a program**

```bash
gcc prog_<q>.c -o prog_<q>
```

Replace `<q>` with the question number (1–8)

3. **Run the program**

```bash
./prog_<q>
```
